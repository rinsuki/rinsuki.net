直接的に支援する

GitHub Sponsors で支援する

各種国際ブランドのクレジットカードが使えるはずです。

https://github.com/sponsors/rinsuki から支援できます。

請求は USD 建てになるはずです (ので、USD で暮らしている人は為替レートに影響されず、それ以外の通貨で暮らしている人は為替レートに影響されて若干変動するはずです)

メリット

  • 個人アカウントから支援する場合、GitHub Sponsors のプラットフォーム手数料がかかりません (クレジットカードの海外手数料等別の部分ではかかることがあります)
  • GitHub が私の口座への振り込み手数料を負担してくれるため (支援金額から天引きではない)、あなたが GitHub Sponsors で支援することで、GitHub に間接的に金を出させることができます (その分のお金を貰うのは私ではないですが)
  • あなたの GitHub プロフィールの Sponsoring 欄に私のプロフィールへのリンクが出せたり、私の GitHub の Sponsors 欄にあなたのプロフィールへのリンクを出せたりします (隠すこともできます)
  • 5 USD 以上の任意の金額を指定できます

デメリット

  • 支援に GitHub アカウントが必要です
  • 公開・非公開に関係なく、私には GitHub アカウントがバレます
  • pixiv FANBOX で読める、だいたい月間で更新される活動報告記事は GitHub Sponsors では (今のところ) 読めません

pixiv FANBOX で支援する

各種国際ブランドのクレジットカードやPayPalアカウントが使えるはずです。

https://rinsuki.fanbox.cc/ から支援できます。

請求は JPY 建てになるはずです (ので、JPY で暮らしている人は為替レートに影響されず、それ以外の通貨で暮らしている人は為替レートに影響されて若干変動するはずです)

メリット

  • だいたい月間で更新される活動報告記事が読めます (今のところ pixiv FANBOX 限定ですが、将来的には予告なく他プラットフォームでも公開する可能性があります)s

デメリット

  • あなたが支払った金額の10%+振込手数料が、支援金から天引きされます (公式ヘルプ)
  • 支援に pixiv アカウントが必要です
  • 公開・非公開に関係なく、私には pixiv アカウントがバレます
  • あなたが私を支援していることを、対外的に証明可能な方法で公開する方法がありません
  • いくつかある金額設定の中から1つを選んで支援することしかできないので、任意金額での支援はできません
